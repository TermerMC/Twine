Configuring Twine

Configuration of Twine can be done by editing twine.yml.

The most basic thing to configure is changing the server's ip and port in the server block.
Many other configuration options including virtualhost domains are available in that file. All options are explained within the file itself, so to get started, just open the file in a text editor and read over it.
